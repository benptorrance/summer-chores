# Summer Chores

This project is an exercise in implementing the three different types of Asynchronous Javascript; Callback functions, Promises, and finally Async and Await. Each script is named according to what method is used to implement a person's routine of performing their summer chores. A list of five chores must be performed by this person, but each chore they complete makes them tired and their is a higher chance that they will fall asleep before completing the remaining chores.

# Usage

To run this code you will need to download the Summer Chores project, then run the different scripts in the terminal.

# Requirements

- Node JS
